🐧 別讓面試官發現我是一隻企鵝 (Don't Notice I'm a Penguin)

「面試官覺得這段程式碼有魚腥味，但你的手速快到讓他來不及深究。」

這是一款基於 Web 的無厘頭生存打字遊戲。
你扮演一隻代號 Pingu++ 的企鵝，穿著西裝混入矽谷面試現場。
你的目標是透過狂按鍵盤假裝寫程式，同時克制自己的動物本能，不要從辦公椅上滑下來！

[開始遊玩](https://adaawu.github.io/game-penguin-engineer)

防止企鵝發呆滑落。如果滑鼠或鍵盤不動，企鵝會在 10 秒後滑下椅子。

🐟 空白鍵 (Space)

大絕招！ 遇到懷疑度過高時，吐出一條魚轉移面試官注意力 (全場限用 3 次)。

🏆 勝利與失敗條件

勝利 (錄取)：成功回答 5 個問題 且未被識破。

失敗 (穿幫)：

發呆太久：滑落警報倒數結束 (10秒)，從椅子上滑下來。

懷疑度爆表：面試官懷疑度達到 100%。

⚙️ 核心機制

滑落警報 (Idle Timer)

企鵝生性好動，如果玩家完全不操作 (不打字也不動滑鼠)，企鵝會開始在椅子上扭來扭去。

畫面會出現 10 秒倒數，倒數歸零即判定失敗。

對策：隨時保持手部忙碌！

面試官懷疑度 (Suspicion)

懷疑度會隨著時間緩慢自然上升。

如果企鵝坐不穩 (發呆太久)，懷疑度會加速上升。

對策：快速打字完成問題回答，每次成功回答問題，懷疑度會大幅下降 (-20%)。

無厘頭問答系統

面試官會詢問各種荒謬的技術問題 (例如：「如何反轉二元樹並解釋你為什麼沒穿褲子？」)。

螢幕上的程式碼是隨機生成的技術廢話 (包含 import fish; 或 git push origin antarctica)。

🛠️ 技術棧 (Tech Stack)

這是一個單一文件的網頁遊戲，無需伺服器即可運行。

Core: HTML5, Vanilla JavaScript (ES6+)

Rendering: HTML5 Canvas API (繪製企鵝、粒子效果、UI)

Styling: Tailwind CSS (via CDN)

Fonts: Google Fonts (Noto Sans TC, VT323)

🚀 如何執行

下載 penguin_interview.html 檔案。

直接使用瀏覽器 (Chrome, Edge, Firefox, Safari) 開啟該檔案。

確保電腦已連上網路 (以載入字體與 CSS 樣式庫)。

點擊「開始面試」即可遊玩。

🎨 美術設計

遊戲內所有圖形（企鵝、辦公室背景、粒子效果）皆使用 Canvas API 純程式碼繪製，沒有使用任何外部圖片素材，確保了極致的輕量化與載入速度。
